# JustMove
### Commuters best friend!

Keeps track of movements in the city, will help you minimize your time spent travelling in your daily life.

### Sensors: 
  - GPS 
  - Accelerometer


### API: 
  - Google Maps API
  - PubliBike API


### Data: 
  - Stored Locally 

### Statistics: 
  - steps 
  - distance
  - time
